# Лабораторийн ажил: Git Clone, Commit, Push, Pull ашиглах

## 1. Түлхүүр уг

Git, GitHub, repository, clone, commit, push, pull, remote repository, local repository, version control

## 2. Хичээлийн зорилго

Оюутнуудад GitHub дээр байрлах repository-г локаль компьютер дээр clone хийх, локаль дээр еерчлелт хийх, commit хийх, GitHub pyy push хийх болон GitHub дээрх шинэ
еерчлелтийг pull хийх процессыг ойлгуулах.

## 3. Суралцах үр дүн

 Энэхүү лабораторийн ажлыг хийсний дараа оюутан дараах чадваруудыг эзэмшинэ.

1. GitHub repository-г локаль компьютерт clone хийх
2. Локаль repository дээр файл өөрчлөх
3. git add болон git commit ашиглах
4. Өөрчлелтүүдийг GitHub pyy git push ашиглан илгээх
5. GitHub дээрх шинэ еерчлелтийг git pull ашиглан татах
